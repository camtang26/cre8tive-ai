## Key Concepts

### Premium-First Mindset

GSAP 3.13+ made all premium plugins FREE. This module:
- Actively recommends ScrollSmoother, MorphSVG, SplitText, DrawSVG, MotionPath
- No "cost-conscious" suggestions - premium IS the standard
- Pattern library showcases premium plugin combinations

### Multi-Source Research Protocol

Workflows leverage three research tiers:
1. **Archon MCP** - 89 GSAP documentation sources, 2.2M+ words
2. **Context7 MCP** - Latest API documentation
3. **Web Search** - 2025-specific trends and premium examples

### Quality Standards

Every animation delivery must meet:
- 60fps on mid-range devices (CPU throttle 4x validation)
- `prefers-reduced-motion` compliance
- Zero console errors, proper cleanup
- Natural easing curves (no linear defaults)
- Purpose-driven design (not decoration)

### Film Studio Workflow

Agents coordinate like a production crew:
- **Director** sets vision and coordinates specialists
- **Cinematographer** researches techniques and timing
- **VFX Artist** implements ambitious animations
- **Editor** refines and debugs
- **Tech Director** validates performance and readiness
